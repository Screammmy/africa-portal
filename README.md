# AfricaPortal — интерактивный шаблон

Кликабельный прототип туристического портала по Африке: лендинг, кабинеты администратора и партнёра.

## Страницы шаблона

- **`index.html`** — кабинеты (админ + партнёр)
- **`landing.html`** — публичный лендинг
- **`portal.html`** — пользовательский портал

## Стек

Чистая статика: HTML + CSS + React (через CDN, без сборки). JSX выполняется в браузере через `@babel/standalone`. Никаких зависимостей и `npm install` для запуска не нужно.

## Локальный запуск

Открыть любой из HTML-файлов в браузере. Для корректной работы загрузки `.jsx` через `fetch` рекомендуется запустить простой статический сервер, например:

```powershell
npx serve .
```

## Структура

```
.
├── index.html          # вход в кабинеты
├── landing.html        # лендинг
├── portal.html         # портал пользователя
├── *.jsx               # React-компоненты (выполняются через Babel в браузере)
├── styles.css          # стили кабинетов
├── landing.css         # стили лендинга
└── docs/               # ТЗ и материалы
```
